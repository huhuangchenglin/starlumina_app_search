# 星芒浏览器起始页

![星芒浏览器](https://via.placeholder.com/150x150.png?text=星芒浏览器)

## 项目概述

星芒浏览器起始页是一个现代化、简洁且功能丰富的浏览器主页应用。它旨在为用户提供一个高效、美观的网络导航入口，集成了多种搜索引擎、快捷访问和个性化设置功能，提升用户的日常浏览体验。

## 主要功能

### 🔍 智能搜索

- **多引擎支持**：集成百度、Google、必应等主流搜索引擎
- **智能识别**：自动识别URL、IP地址和搜索关键词
- **搜索建议**：实时显示搜索建议，提高搜索效率
- **搜索历史**：记录并快速访问最近的搜索内容

### ⚙️ 个性化设置

- **主题切换**：支持明亮模式和深色模式，自动适应系统设置
- **搜索引擎选择**：自定义默认搜索引擎
- **界面定制**：可调整搜索历史显示、界面元素等
- **数据本地存储**：所有设置和历史记录存储在本地，保护隐私

### 🚀 快捷访问

- **Docker栏**：底部固定式快捷启动栏，可访问常用网站和功能
- **可视化图标**：直观的图标设计，提高操作效率
- **流畅动画**：平滑的交互动画，提升用户体验

## 技术特点

- **响应式设计**：完美适配各种设备屏幕尺寸
- **轻量级架构**：快速加载，低资源占用
- **现代UI/UX**：简洁优雅的用户界面，符合当代设计趋势
- **无障碍支持**：考虑各类用户需求，提供无障碍操作体验
- **本地优先**：数据存储在本地，无需云端同步，保护用户隐私

## 安装与使用

1. 克隆或下载本项目到您的Web服务器
2. 确保服务器支持PHP
3. 访问项目根目录下的index.php
4. 可选：将页面设置为浏览器主页或新标签页
